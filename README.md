## 🎯 Project Objective

Develop an intelligent obesity risk classification system capable of predicting obesity categories using demographic, lifestyle, and health-related factors. The project compares multiple Machine Learning and Deep Learning models to identify the most effective solution.

---

## 🚀 Highlights

✔ Data Cleaning & Preprocessing

✔ Outlier Detection using IQR Method

✔ Feature Scaling with StandardScaler

✔ Hyperparameter Tuning using GridSearchCV

✔ Cross Validation

✔ Multi-Class Classification

✔ Artificial Neural Network (ANN)

✔ Model Performance Comparison

✔ Feature Importance Analysis

## 🏆 Results

| Model         | Train Accuracy | Test Accuracy |
| ------------- | -------------- | ------------- |
| SVC           | 89%            | 88%           |
| KNN           | 80%            | 74%           |
| Random Forest | 88%            | 85%           |
| ANN           | 81%            | 76%           |

"Random forest classifier" final best-performing model.
